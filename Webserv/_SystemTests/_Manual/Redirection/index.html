Webserv HTTP Redirection Tests

HTTP Redirect Status Codes

Code Name Description Method Handling Caching
301 Moved Permanently Resource has been permanently moved to a new URL May convert POST to GET Cacheable
302 Found (Temporary Redirect) Resource temporarily located at a different URL May convert POST to GET Not cacheable
303 See Other Response to request can be found at another URL using GET Always use GET for the new location Not cacheable
307 Temporary Redirect Same as 302 but strictly preserves method Preserves original method Not cacheable
308 Permanent Redirect Same as 301 but strictly preserves method Preserves original method Cacheable
