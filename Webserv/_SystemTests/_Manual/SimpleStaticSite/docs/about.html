About the Webserv Project

Webserv is a project at 42 that challenges students to create an HTTP server from scratch. The goal is to implement a server that can handle HTTP/1.1 requests and responses, similar to Nginx or Apache.

Key features implemented in our Webserv:

  • HTTP/1.1 protocol compliance
  • Multiple virtual servers
  • Directory listing
  • HTTP redirections
  • File uploads
  • Custom error pages
  • CGI execution

This project has been a great learning experience, teaching us about networking, HTTP protocols, system programming, and more.
