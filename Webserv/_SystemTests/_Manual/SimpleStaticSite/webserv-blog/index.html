Welcome to Our Webserv Project

42 School Project | Tags: webserv http c++

Welcome to the demonstration site for our custom HTTP/1.1 server implementation. This static site showcases the capabilities of our server developed as part of the 42 curriculum by a team of three students.

Webserv Features

  • HTTP/1.1 Compliance - Standard-compliant request and response handling
  • Multiple Virtual Servers - Support for different server configurations
  • Static File Serving - Efficiently serving various file types
  • Directory Listing - Auto-indexing of directories
  • HTTP Redirections - Support for 301, 302, 307, and 308 redirects
  • Custom Error Pages - Configurable error responses
  • File Uploads - Support for multipart form data
  • CGI Execution - Dynamic content generation via script interpreters
  • Configurable Limits - Control resource usage with configurable limits

Learn more about server configuration options and how to set up your own Webserv instance.
