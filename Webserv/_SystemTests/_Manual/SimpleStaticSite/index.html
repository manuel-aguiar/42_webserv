Welcome to My Webserv Blog

Welcome to my personal blog powered by Webserv! This is a simple static site that demonstrates the capabilities of our custom HTTP server implementation. The server supports directory listing, static file serving, and various other HTTP features.

This blog is designed to showcase how you can create a simple yet effective personal website using our Webserv project. Feel free to explore the different sections and features.

Directory Listing Feature

Posted on March 6, 2023 | Tags: directory-listing feature

One of the key features of Webserv is directory listing. When enabled using the auto_index 1 directive in the server configuration, Webserv will generate a list of files and directories when no index file is present.

You can see this feature in action by visiting the following directories:

  • Images Directory - Browse images and media files
  • Documents Directory - Browse documentation files

The directory listing feature is particularly useful for sharing files with others or creating a simple file browser interface without the need for complex server-side scripting.

HTTP Redirection in Webserv

Posted on March 5, 2023 | Tags: redirection http

Webserv implements HTTP redirection according to the standards. We support various redirection status codes:

  • 301 - Moved Permanently
  • 302 - Found (Temporary Redirect)
  • 307 - Temporary Redirect (preserves method)
  • 308 - Permanent Redirect (preserves method)

Check out these examples:

  • Redirect to Images (301)
  • Redirect to Documents (302)
  • External Redirect (42.fr)

About This Blog
